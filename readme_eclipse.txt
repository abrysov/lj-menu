Setup project in eclipse:

1. Import Existing Maven Project "menu-pom"

2. Go to menu-common-lj Properties -> Java Build Path -> Order and Export and make sure you have the same listing there:

	menu-common-lj/gin
	Android 4.2.2 (checked)
	Android Dependencies (checked)
	Android Private Libraries (checked)
	Maven Dependencies (checked)
	menu-common-lj/src

3. You may need to build several times.
